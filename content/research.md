### Research Interests

I'm interested in helping programmers build **provably correct, efficient** software, using techniques from [programming languages](https://en.wikipedia.org/wiki/Programming_language_theory) and [formal methods](https://en.wikipedia.org/wiki/Formal_methods). To achieve this, I work on:
- developing *program synthesizers* that
    generate correct-by-construction software automatically
- *formally verifying* that existing software
    conforms to logical specifications
- designing *domain-specific languages* to make it easier to build reliable software for
    emerging domains.
